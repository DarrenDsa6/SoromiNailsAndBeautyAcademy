# Soromi Nails and Beauty Academy

## Frontend Documentation

This documentation outlines the structure and usage of the static frontend for the Soromi Nails and Beauty Academy website.

### Overview
The static website is designed to provide information about our nail and beauty services, allowing users to easily navigate and learn more about what we offer. It is optimized for both desktop and mobile views.

### Folder Structure
- `index.html`: The main entry point of the website.
- `css/`: Contains all the stylesheets.
  - `styles.css`: Primary stylesheet for the website.
- `images/`: Contains images used throughout the website.
- `js/`: Contains JavaScript files for any interactivity.
